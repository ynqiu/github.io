如何建立Hexo博客

发表于 2018-02-28 |

前言

之前一直使用Jekyll和Github Pages来建立个人博客，但是发现无论我如何找，都找不到一个合适的Jekyll模板。直到最后发现别人用Hexo搭建的博客网站，才发现Hexo相比Jekyll可以更加优雅！遂抛弃之前的Jekyll转投Hexo，并且在强大的Next主题加持下，建成了现在的网站模样。

由于Hexo搭建博客的教程已经非常成熟，本人用了半天的时间，参考了众多网友的博客及教程，暂且成功地使用了这个博客网站。因此在这里，本人不再重新写教程(因为太多重复了，难免混淆视听)，在这里本人仅仅提供一些主要参考的博客内容，为日后读者查找资料提供便利。
